# GitHub Marketplace Submission Checklist

This document tracks the requirements for submitting AgentProof PR to the GitHub Marketplace.

## 📝 Description
- [ ] **Short Tagline:** (Max 50 chars) Stop reviewing half-finished AI-generated pull requests.
- [ ] **Long Description:** 
  AgentProof PR is a GitHub Action that automatically scores pull requests for human-readiness. It ensures that PRs—especially those generated by AI coding agents—meet your team's quality standards before a human developer ever lays eyes on them.
  
  Stop the cycle of "nitpick" reviews by automating the basic hygiene checks: linked tickets, descriptive summaries, test evidence, and clean code (no TODOs or debug logs).

## 🔗 Links
- [ ] **Support Link:** https://github.com/mattfred/agentproof-pr/issues
- [ ] **Privacy Policy:** 
  AgentProof PR runs entirely within your GitHub Actions environment. No code, metadata, or PR content is ever sent to external servers. Your data stays within your GitHub infrastructure.
- [ ] **Installation Instructions:** 
  1. Add a `.github/workflows/agentproof.yml` file to your repository.
  2. (Optional) Customize rules by adding a `.agentproof.yml` file to your repository root.
  3. See the [Quick Start](https://github.com/mattfred/agentproof-pr#quick-start) for details.
