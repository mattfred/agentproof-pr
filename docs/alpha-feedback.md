# Alpha Feedback

Thank you for participating in the AgentProof PR private alpha! Your feedback is crucial for making this tool better for everyone.

## Alpha Feedback Checklist

Please copy this checklist and include it in your feedback (via GitHub Issue or direct message):

- [ ] **Was setup easy?** (Did it take under 5 minutes?)
- [ ] **Did the score feel fair?** (Did high-quality PRs get high scores?)
- [ ] **Were warnings actionable?** (Was it clear what needed to be fixed?)
- [ ] **Did it reduce reviewer time?** (Did humans spend less time on trivial checks?)
- [ ] **Which rules were too strict?** (Are any rules too hard to pass?)
- [ ] **Which rules were too noisy?** (Are any rules giving false positives?)
- [ ] **Would you pay for this?** (Is the value proposition clear?)

## How to Provide Feedback

1. **GitHub Issues**: Open a [New Issue](https://github.com/mattfred/agentproof-pr/issues) with the tag `alpha-feedback`.
2. **Discussions**: Start a thread in our [Discussions](https://github.com/mattfred/agentproof-pr/discussions) area.

## Feature Requests

If you have ideas for new rules, integrations, or improvements, we'd love to hear them!
