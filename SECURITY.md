# Security Policy

## Supported Versions

Currently, only the latest version of AgentProof PR is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within AgentProof PR, please open an issue or contact the maintainers directly.

We aim to acknowledge all reports within 48 hours and provide a fix or mitigation within 5 business days.

## Privacy & Security Note

AgentProof PR runs as a GitHub Action. It only accesses the data provided by the GitHub Actions environment and the content of the pull request it is running on. No code or data is sent to external servers by AgentProof PR itself.
